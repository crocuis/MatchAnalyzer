# AGENTS.md

이 문서는 이 저장소에서 작업하는 에이전트와 기여자가 반드시 따라야 하는 실행 규칙만 정의한다.

## 1. 필수 도구 / 명령

- 텍스트 검색은 반드시 `rg`를 사용한다. 내용 검색에 `grep`, 파일 목록 검색에 `find`를 기본 대체 수단으로 쓰지 않는다.
- Python 테스트 실행은 반드시 `python3 -m pytest`를 사용한다. `pytest` 단독 호출로 대체하지 않는다.
- Node 테스트 실행은 반드시 `npm` 기준으로 수행한다. 임의의 직접 실행(`npx vitest`, `pnpm`, `yarn`)으로 대체하지 않는다.
- 변경 검토는 반드시 `git status --short`, `git diff --stat`, `git diff --check`로 확인한다.
- 전체 성공 주장 전에는 실제 명령을 실행해 결과를 확인한다. 추정으로 “통과했을 것”이라고 쓰지 않는다.

## 2. 테스트 순서와 통과 기준

아래 순서대로 실행한다. 해당 경로/패키지가 아직 없으면 `N/A`로 명시하고 넘어간다.

1. `npm test`
2. `npm --workspace apps/api run test`
3. `npm --workspace apps/web run test`
4. `python3 -m pytest`

통과 기준:

- 실행한 테스트 명령이 모두 종료 코드 `0`이어야 한다.
- 실패, skip, xfail은 구분해서 기록한다. skip이 있으면 이유를 남긴다.
- 경고가 남더라도 기능/보안/성능에 영향이 있으면 통과로 간주하지 않는다.
- 테스트를 못 돌렸으면 완료 보고를 하지 말고, 막힌 이유를 먼저 적는다.

## 3. 보안 최소 가드레일

- 비밀값, 토큰, 키, 세션 정보는 코드, 문서, 커밋에 남기지 않는다.
- 클라이언트 코드에는 서버 비밀값을 넣지 않는다.
- 외부 입력은 저장 전 검증하고, 원천 페이로드와 정규화 결과를 혼동하지 않는다.
- 예측 생성의 정본 경로는 서버/배치에 둔다. 클라이언트 계산 결과는 검증 후에만 채택한다.
- 로그에는 개인정보, 인증값, 원시 자격 증명을 남기지 않는다.

## 4. 성능 최소 가드레일

- 모델 학습과 정본 예측 추론은 요청 경로에서 실행하지 않는다.
- 무거운 계산은 배치에서 미리 계산하고, 읽기 경로는 조회와 캐시에 집중한다.
- 큰 원천 페이로드는 오브젝트 스토리지에 두고, 관계형 DB에는 조회용 정규화 결과만 저장한다.
- 응답은 필요한 필드만 반환한다. 불필요한 대량 필드, 전체 row fetch, 과한 join을 기본값으로 두지 않는다.
- 성능 문제가 의심되면 먼저 페이로드 축소, 쿼리 축소, 캐시 개선을 하고, 그 다음에만 언어/런타임 교체를 검토한다.

## 5. PR 전 필수 체크 명령

PR 또는 최종 제출 전 반드시 아래를 순서대로 실행한다.

1. `git status --short`
2. `git diff --stat`
3. `git diff --check`
4. `npm test`
5. `npm --workspace apps/api run test`
6. `npm --workspace apps/web run test`
7. `python3 -m pytest`

기록 규칙:

- 어떤 명령을 실행했는지 적는다.
- 실패한 명령이 있으면 숨기지 말고 그대로 적는다.
- 실행하지 못한 명령은 이유와 함께 `미실행`으로 적는다.

## 6. 코드 / 문서 작성 규칙

- 코드의 식별자, 함수명, 타입명, 파일명, 테스트명은 영문으로 작성한다.
- 커밋 메시지는 한글로 작성한다.
- 문서 본문은 한글로 작성한다.
- 코드 블록 안 예시는 실제 구현에 가까운 영문 코드로 작성한다.
- 주석은 꼭 필요할 때만 추가하고, 이유와 제약을 설명하는 짧은 한글 주석을 우선한다.
- 한 파일에는 한 가지 주된 책임만 둔다. 큰 파일을 더 키우기보다 경계를 먼저 검토한다.
- 새 의존성은 명확한 필요가 있을 때만 추가한다.
- 플레이스홀더(`TODO`, `TBD`, “나중에 구현”)를 남긴 채 완료 처리하지 않는다.
- 장황한 배경 설명보다 실행 규칙, 검증 결과, 남은 리스크를 우선 기록한다.
